# JobTracer 飞书 Bot 操作手册

JobTracer 通过飞书 Bot 向你推送求职相关的通知，你可以通过 Bot 进行交互操作。

---

## 📬 你会收到哪些通知？

### 1. 📄 简历已生成
当数字足迹扫描完成并生成简历后，Bot 会发送通知：

```
📄 简历生成完成

目标职位：Python 后端工程师
项目数量：5 个
技能标签：Python / Redis / 微服务 / Docker

[查看简历]  [修改目标职位]
```

### 2. 🎯 新职位发现
当发现高匹配度的新职位时，Bot 会推送：

```
🎯 发现新职位 - 匹配度 92%

某科技有限公司 | Python 后端工程师
薪资：25-40K | 经验：3-5年 | 城市：北京
标签：五险一金 🎁 弹性工作 ⏰ 股票期权 💰

[查看详情]  [立即投递]  [稍后处理]
```

### 3. 📊 每日求职日报
每天固定时间推送求职进度汇总：

```
📊 今日求职日报 - 2024-01-01

投递统计：
• 今日投递：3 个
• 累计投递：15 个
• HR 回复：2 个
• 面试邀请：1 个

近期投递：
• 某科技 - Python 后端 ✅ 已投递
• 某电商 - Go 工程师 ⏳ 查看中
• 某金融 - 资深开发 🔔 HR 回复

[查看全部投递]  [职位推荐]
```

### 4. 💬 HR 回复提醒
当 HR 回复你的消息时，Bot 会立即通知：

```
💬 HR 回复了！

公司：某科技有限公司
职位：Python 后端工程师
回复：「您好，我们正在看您的简历，方便的话可以约个时间面试吗？」

[查看对话]  [预约面试]  [标记已读]
```

## 🔘 按钮操作说明

### [查看详情]
- 查看职位的完整 JD（职位描述）
- 查看公司信息、团队规模、技术栈要求
- 查看与你的匹配度分析

### [立即投递] / [投递]
- 确认后立即向该职位发送招呼语
- 系统会自动生成定制化的招呼语
- 发送成功后会记录到投递列表

### [稍后处理]
- 将职位加入「待处理」列表
- 之后可以在 [我的投递] 中找到
- 不会立即发送招呼

### [一键投递]
- 批量投递多个待处理职位
- 适用于集中投递场景

### [复制招呼语]
- 将生成的招呼语复制到剪贴板
- 方便你手动粘贴到 BOSS 直聘 App

### [修改简历]
- 调整简历中的项目选择
- 修改技能权重
- 更新目标职位

### [查看简历]
- 查看当前生成的完整简历
- 可以导出或分享

## 📋 状态说明

| 状态 | 图标 | 含义 |
|------|------|------|
| 待处理 | ⏳ | 等待你确认投递 |
| 已投递 | ✅ | 已经发送招呼 |
| 查看中 | 👀 | HR 正在查看 |
| 已回复 | 💬 | HR 有新消息 |
| 已拒绝 | ❌ | 不合适 |
| 已过期 | ⏰ | 职位已下架 |
| 面试中 | 📅 | 进入面试流程 |

## 🔄 常用操作

### 查看我的投递
发送：`我的投递` 或 `投递列表`

### 查看职位推荐
发送：`推荐职位` 或 `找职位`

### 重新生成招呼语
在职位详情页点击 [重新发送]

### 修改目标职位
发送：`修改目标 [职位名称]`

### 暂停通知
发送：`暂停通知` 或 `勿扰模式`

---

## 💡 常见问题

**Q: 为什么没收到通知？**
- 检查飞书 Bot 是否已启用
- 确认没有开启「静音」或「勿扰」模式

**Q: 投递失败怎么办？**
- 检查 BOSS Cookie 是否过期
- 点击 [重新发送] 重试

**Q: 如何取消订阅？**
- 发送 `取消订阅` 给 Bot
